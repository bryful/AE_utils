# AE_Menu
AEのscriptで簡単にメニューランチャー作るとフォントの指定ができないので見ずらい。
かと言ってCEPでパッケージ作るのも面倒なので作ったメニュー作成アプリです。  
  
ボタンをテキストではなくiconbuttonで画像で表示させます。  
  
## 使い方
適当にスクリプトをまとめたフォルダを作ります。scriptsフォルダ内に作る場合は()でくくって読み込まないようにしておくと楽です。
そのフォルダをこのアプリにドラッグ&ドロップするとスクリプトファイルがメニューとして表示されます。
右クリックしてExportScriptを実行すると、それらのスクリプトを呼び出すメニュースクリプトが書き出されます。  
  
スクリプトは絶対パスで記述されているので、メニュースクリプトを"scripts/ScriptUI Panels"にコピーしてAEを再起動すればインストールできます。
